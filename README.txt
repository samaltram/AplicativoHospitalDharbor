Este proyecto consta de dos partes

BackEnd realisado en JAVA (Sprint) fue realizado en el ID STS (Sprint Tool Suite 4)

FrontEnd realisado en Anagular 13 (material, bootstrap, popper.js) fue realizado en el ID VS Code

Se utilizo la base de datos MySQL el nombre de la base de datos es hospital

Instalacion

0. Clonar o desempaquetar el Archivo en el Directorio de su preferencia
CREADO BASE DE DATOS
1. Crear la Base de Datos en MySQL con el Sript que se encunetra en el directio Base de Datos
EJECUTANDO FRONTEND
2. En ID STS subir cargar el Directorio hospital que se encuntra en directorio BackEnd
3. Configurar el archivo application.properties que se encuentra en la ruta 
	hospital/src/main/resurces
datos actuales
	spring.datasource.url=jdbc:mysql://root:ars.Leumas008@localhost:3306/hospital
	spring.datasource.username=root
	spring.datasource.password=ars.Leumas008
cambiar credenciales

4. en Boot Dashboard de STS
   Ejecutar de manera local vale decir http://localhost:8080

EJECUTANDO FRONTEND

5. En el ID VS code cargar el directorio AppHospital
6. En la Terminal del ID correr > npm i
7. En la Terminal del ID para Ejecutar el proyecto > ng serve
   Esto Ejuctara el proyecto en http://localhost:4200

8. En un navegador para ver el Aplicativo en ejecucion ir a la url http://localhost:4200
9. El usuario y clave son nominativos.
